# User stories

- a user shall be able to register poop log
- a user shall be able to edit their poop log
- a user shall be able to delete their poop log
- a user shall be able to see their poop history by month and year
- a user shall be able to create a poop contest with other users
- a user shall be able to view poop contest's leaderboards
- a user shall be able to view other users totals, by month and year
